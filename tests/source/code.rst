Code Examples
=============

Some code using double colon mode (default language)::

   import hello

   hello.world()

   def main():
       hello.world()

A python code directive:

.. code:: python

   import hello

   def main():
       hello.world()

A C code directive:

.. code:: C

   #include "hello.h"

   int hello_world() {
     printf("Hello, world");
   }
